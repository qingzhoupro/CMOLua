# 高度和深度格式

> 参考: Data_Types.md

## 高度格式

CMO 中高度可以用米或英尺表示。

### 语法

```lua
{altitude = "数值"}          -- 默认米
{altitude = "数值 M"}        -- 明确米
{altitude = "数值 FT"}        -- 英尺
```

### 示例

```lua
-- 米（默认）
{altitude = "5000"}      -- 5000 米
{altitude = "10000"}     -- 10000 米

-- 英尺
{altitude = "15000 FT"}   -- 15000 英尺 ≈ 4572 米

-- 负值（相对高度）
{altitude = "-100"}       -- 相对高度 -100 米
```

### 常见高度参考

| 用途 | 高度（米） | 高度（英尺） |
|------|-----------|-------------|
| 超低空 | 100 | 330 |
| 低空 | 500 | 1650 |
| 中空 | 3000 | 10000 |
| 高空 | 7000 | 23000 |
| 超高空 | 12000 | 40000 |

## 深度格式（潜艇）

潜艇使用 `manualAltitude`，正数表示下潜深度。

### 语法

```lua
{manualAltitude = 数值}  -- 正数 = 下潜深度
```

### 示例

```lua
-- 潜艇下潜
{type = "Submarine", manualAltitude = 50}    -- 50 米深度
{type = "Submarine", manualAltitude = 100}   -- 100 米深度
{type = "Submarine", manualAltitude = 200}    -- 200 米深度
```

### 常见深度参考

| 深度类型 | 深度（米） |
|---------|-----------|
| 通气管深度 | 10-15 |
| 安全深度 | 30-50 |
| 巡逻深度 | 50-100 |
| 标准深度 | 100-150 |
| 深潜 | 200-300 |
| 最大深度 | 500+ |

---

## AI 使用提示

1. **飞机必须设置高度**：不使用 altitude 参数会导致错误
2. **潜艇使用 manualAltitude**：不是 altitude
3. **高度单位**：未指定时默认**米**，使用 FT 后缀表示英尺
4. **深度正数**：下潜深度必须是正数
5. **altitude vs altitude FT**：纯数字默认米；`"5000 FT"` 表示英尺
