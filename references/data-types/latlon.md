# 经纬度格式

> 参考: Data_Types.md

## 纬度格式 (Latitude)

### 语法

```lua
{latitude = "值"}
```

### 格式选项

| 格式 | 示例 | 说明 |
|------|------|------|
| 十进制 | `"35.5"` | 正数=北纬，负数=南纬 |
| 带方向 | `"N 35.5"` | N=北纬，S=南纬 |
| DMS 格式 | `"N 35.30.15"` | 度.分.秒 |

### 示例

```lua
-- 十进制（推荐）
{latitude = "35.5"}      -- 北纬 35.5°
{latitude = "-35.5"}     -- 南纬 35.5°

-- 带方向
{latitude = "N 35.5"}    -- 北纬 35.5°
{latitude = "S 35.5"}    -- 南纬 35.5°
```

---

## 经度格式 (Longitude)

### 语法

```lua
{longitude = "值"}
```

### 格式选项

| 格式 | 示例 | 说明 |
|------|------|------|
| 十进制 | `"127.0"` | 正数=东经，负数=西经 |
| 带方向 | `"E 127.0"` | E=东经，W=西经 |
| DMS 格式 | `"E 127.30.15"` | 度.分.秒 |

### 示例

```lua
-- 十进制（推荐）
{longitude = "127.0"}    -- 东经 127.0°
{longitude = "-127.0"}   -- 西经 127.0°

-- 带方向
{longitude = "E 127.0"}  -- 东经 127.0°
{longitude = "W 127.0"}   -- 西经 127.0°
```

---

## 组合使用

```lua
-- 朝鲜半岛（示例位置）
{latitude = "37.5", longitude = "127.0"}

-- 日本海
{latitude = "N 37.5", longitude = "E 140.0"}

-- 美国西海岸
{latitude = "33.0", longitude = "-118.0"}

-- 地中海
{latitude = "35.0", longitude = "15.0"}
```

## AI 使用提示

1. **优先使用十进制**：最简洁，如 `"35.5"`
2. **经纬度都是字符串**：用引号包裹
3. **方向约定**：
   - 纬度：正数=北，负数=南
   - 经度：正数=东，负数=西
4. **精度**：通常 1-2 位小数足够
5. **官方别名**：`lat` / `lon` 是 `latitude` / `longitude` 的官方简写，效果完全相同，无需强制使用完整单词
