# 导弹特征修改案例 - 代码解析

## 特征参数说明

### 雷达散射截面（RCS）

| 参数 | 说明 | 单位 |
|------|------|------|
| RCSFront | 前向 RCS | dBsm |
| RCSRear | 后向 RCS | dBsm |
| RCSSide | 侧向 RCS | dBsm |
| RCSTop | 顶部 RCS | dBsm |

### 红外特征

| 参数 | 说明 |
|------|------|
| IRFront | 前向红外 |
| IRRear | 后向红外 |
| IRSide | 侧向红外 |
| IRTop | 顶部红外 |

## 商业版替代方案

CMO 商业版无法读取 CSV，可考虑：

1. **预设特征**：在编辑器中直接设置导弹特征
2. **简化版本**：硬编码少量特征参数
3. **开发版测试**：仅在开发版调试想定

时间,pitch,yaw,roll,lon,lat,alt,RCS_Top,RCS_Rear,RCS_Front,RCS_Side,...
