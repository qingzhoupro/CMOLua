# 导弹特征修改

## 场景描述

修改导弹的红外/雷达特征参数，实现：
- 雷达散射截面（RCS）调整
- 红外特征参数设置
- 导弹飞行姿态控制

## ⚠️ 重要限制

- **CMO 商业版不支持 Lua IO 操作**，无法读取外部 CSV 文件
- 本案例仅供 **CMO 开发版/服务器版** 用户参考
- 商业版用户需要通过其他方式预设导弹特征

## 适用场景

- 导弹特征模拟研究
- 隐身性能测试
- 电子对抗演练
- 想定调试（开发版）

## 前置条件

- CMO 开发版或服务器版
- 已准备导弹特征数据 CSV 文件
- 已知目标导弹 DBID

## 核心原理

- `ScenEdit_GetUnit()` 按 DBID 获取导弹单位
- `unit.xsection` 设置雷达/红外特征
- 定时事件更新导弹位置和姿态
