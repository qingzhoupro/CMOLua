# 机场自动生成案例 - 代码解析

## group 属性的作用

`u.group = basename` 将所有机场设施归入同一组，在编辑器中可整体选中、移动或删除。

## DBID 说明

各设施的 DBID 随 CMO 数据库版本变化，建议：

```lua
-- 使用 MCP 查询（AI 助手可调用）
query_dbid("跑道")
query_dbid("滑行道")
query_dbid("停机坪")
```

## 经纬度偏移逻辑

每个设施沿纬度方向依次偏移 `.004` 度（约 400-500 米），确保设施不重叠：

```
[跑道] → lat += 0.004
[滑行道] → lon += 0.002
[进入点] → lon += 0.004
[停机坪] → lon += 0.006
[弹药库] → lon += 0.008
```

## 进阶建议

1. **传入设施位置参数**：可增加各设施相对基地的偏移参数
2. **跑道朝向**：通过设置 `heading` 参数控制跑道方向
3. **并行创建**：不同设施可并行创建，节省时间
