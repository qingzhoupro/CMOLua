# 心理战：随机劝降敌方单位

> 📖 **配套文章**：[俄乌冲突-马里乌波尔决战（心理战场景）](https://mp.weixin.qq.com/s/jo0JHSDHdLvVqSZov3mQOA)（公众号：海空兵棋与AI）

## 场景描述

模拟心理战效果：
1. 扫描敌方单位名称
2. 筛选包含特定关键词的单元
3. 随机抽取若干单元使其叛变（转为平民阵营）

> 公众号文章以 2022 年马里乌波尔之战为背景，首次在 CMO 中融入网络战、心理战、舆论战等认知域作战元素。俄军通过心理战分队向前线官兵推送乌军指挥官黑料，成功动摇第 21 旅部分官兵抵抗决心，实现"不战而屈人之兵"。

## 适用场景

- 心理战（PSYOP）演练
- 模拟士气崩溃效果
- 敌方单位哗变场景
- 宣传攻势效果演示

## 前置条件

- 场景中存在可被劝降的敌方单位
- 目标单位名称包含特定识别关键词
- 存在 `Civilian` 阵营（或可接受的目标阵营）

## 核心 API

| 函数 | 说明 |
|------|------|
| `VP_GetSide()` | 获取某阵营所有单位 |
| `ScenEdit_GetUnit({guid=})` | 获取单位详细信息 |
| `string.match()` | 按关键词筛选目标（模糊匹配） |
| `math.random(n)` | 随机返回 1~n 的整数 |
| `ScenEdit_SetUnitSide()` | 改变单位阵营 |
