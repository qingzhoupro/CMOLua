# 网电作战：网络中断与恢复

> 📖 **配套文章**：[【教程系列】通信干扰？](https://mp.weixin.qq.com/s/n0aNY_MJbHIADN8rjfTYpQ)（公众号：海空兵棋与AI）

## 场景描述

模拟网络/电子攻击效果：
1. 中断敌方通信网络（指定单位 `outOfComms = true`）
2. 恢复通信（指定单位 `outOfComms = false`）
3. 自动恢复（通过时间事件触发）

> 公众号文章系统讲解了 CMO 中"通讯中断"的 7 种触发方式（通信设备损坏、潜艇下潜、卫星离线、网络攻击、C3I 节点被毁等），以及"离线"后单位的实际表现：失去共享态势、只能依靠自身传感器感知、不受指挥等。还分析了俄乌冲突中俄军对乌军雷达站的网络攻击案例。

## 适用场景

- 电子战（EW）场景模拟
- 网络中心战演练
- 通信系统攻防
- 想定中临时断网效果演示

## 前置条件

- 目标阵营存在
- 需中断的单位名称需与场景中实际名称匹配

## 核心 API

| 函数 | 说明 |
|------|------|
| `ScenEdit_SetUnit({outOfComms="True"})` | 中断指定单位的网络连接 |
| `ScenEdit_SetUnit({outOfComms="False"})` | 恢复网络连接 |
| `ScenEdit_MsgBox(msg, type)` | 显示消息框 |
| `ScenEdit_SetEvent(name, {IsActive=})` | 启用/禁用事件 |
