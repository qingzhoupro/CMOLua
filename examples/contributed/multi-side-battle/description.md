# 三方对阵：红、蓝、绿三方势力

## 场景描述

创建一个包含三个阵营的复杂战场：
- **Red（红色）**：敌对势力
- **Blue（蓝色）**：我方友军
- **Green（绿色）**：中立/观察者

## 适用场景

- 多方冲突演练
- 中立方干预场景
- 复杂外交关系模拟
- 联军作战想定

## 核心 API

| 函数 | 说明 |
|------|------|
| `ScenEdit_AddSide()` | 创建阵营 |
| `ScenEdit_SetSideOptions()` | 设置阵营属性 |
| `ScenEdit_SetSidePosture()` | 设置阵营关系（双向） |
