# 亚太主要海峡参考点

## 场景描述

为亚太地区主要战略海峡创建参考点区域，便于快速设置巡逻区域、布雷区、封锁区等。

## 适用场景

- 海上交通线（SLOC）封锁演练
- 海峡通道巡逻任务
- 反潜巡逻区域设置
- 布雷封锁区划设
